# Micro-Spec Template

## Task

- 一句话描述当前任务

## Task Understanding

- 当前对任务的理解

## Goal

- 本轮目标

## Scope

- 本轮明确要覆盖的范围

## Constraints

- 当前约束与边界

## Approval Status

- `Pending Approval` / `Plan Approved` / `Executed`

## Touched Files

- 预计涉及哪些文件

## Risk

- 主要风险是什么

## Done Contract

- 什么算完成
- 由什么证明完成

## Validation

- 用什么测试、日志或人工验证来确认结果

## Truth Docs Impact

- 涉及哪些真相文档
- 哪些无需更正，为什么
- 哪些已经更正，或仍待更正

## Next Actions

- 下一步 1-3 个动作

## Change Log

- 执行后回写改动结论

## Resume or Handoff

- 如果暂停，下一轮目标与恢复锚点是什么
